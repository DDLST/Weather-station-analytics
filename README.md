# Мини-система сбора и анализа данных (Погодная станция)

Система генерирует осмысленные показания “погодных станций”, сохраняет в MySQL и позволяет анализировать в Redash.

## Компоненты
- MySQL — хранит станции и показания (`station`, `reading`)
- Python-генератор — автоматически пишет данные 
- Redash — визуализации и дашборд

## Требования
- Docker + Docker Desktop
- Свободные порты: 80, 3306, 5000

## Запуск
1) Скопировать env:
- `.env.example` → `.env`
- заменить пароли/секреты

2) Поднять контейнеры:
```bash
docker-compose up -d --build
